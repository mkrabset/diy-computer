Output registers and displays

outreg_display.fzz   -   Single byte display, can be hooked up with x,y or z register

outregx3_v2_registers.fzz  -  3 x 1-byte output register

outregx3_v2.fzz  -  Display for outregx3_v2_registers, can be stacked on top

The 7-segment displays are common anode (5611BH)
